### A word about options
Most functions take a options argument which accepts an Object.
This is the default mode for each function
```JS
const defaultMode = {
  hardFail: false,
  complex: true,
  complex3: false,
  complexCho: true,
  complexJung: true,
  complexJong: true,
  complexArchaic: false,
  composeComplexDouble: true,
  decomposeComplexDouble: false,
  grouped: false,
};
```
